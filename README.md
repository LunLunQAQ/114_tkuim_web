## 專案簡介
實用的記帳網頁，可記錄收入與支出，自動計算餘額並保存資料。

## 使用技術
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- API 表單驗證

## 功能特色

### 核心功能
1. **新增記錄** - 輸入項目名稱、金額、類型
2. **表單驗證** - HTML5 即時驗證與錯誤提示
3. **自動計算** - 即時顯示餘額、總收入、總支出
4. **記錄管理** - 完整交易列表，可刪除單筆記錄
5. **資料保存** - LocalStorage 儲存，重開不遺失


## 功能特色
1. 填寫項目名稱、金額，選擇收入或支出
2. 點擊「新增記錄」按鈕
3. 查看上方統計卡片的即時更新
4. 點擊垃圾桶圖示可刪除記錄
5. 點擊右上角圖示切換深色/淺色模式
